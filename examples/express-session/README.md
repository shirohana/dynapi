## Express-session with Dynapi.js

### Install dependencies

##### Using NPM
```
$ npm install
```

##### Using Yarn
```
$ yarn install
```

### Start server

```
$ node index.js
```

### Done
Now you can visit [http://127.0.0.1:3000](http://127.0.0.1:3000)
