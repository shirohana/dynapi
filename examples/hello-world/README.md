## Hello World with Dynapi.js

### Install dependencies

##### Using NPM
```
$ npm install
```

##### Using yarn
```
$ yarn install
```

### Start server

```
$ node index.js
```

### Done
Now you can visit [http://127.0.0.1:3000][link] and try to modify `./api/post.js`.

[link]: http://127.0.0.1:3000
